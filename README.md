# Wedding Invitation

A simple wedding invitation landing page in Arabic, featuring a polished hero section, animated theme controls, and decorative visuals.

## Project Overview

- **Title:** دعوة زفاف | اسلام & مى
- **Language:** Arabic (RTL)
- **Files:** `index.html`, `old.html`, `assets/`
- **Purpose:** Showcase a wedding invitation design with a modern responsive layout and rich visuals.

## What’s included

- `index.html` — main invitation page with styles and interactions
- `old.html` — alternate version of the invitation
- `assets/` — icons, images, and media resources used by the pages

## How to view

1. Open `index.html` in your browser.
2. Optionally use `old.html` for the older version of the invitation.

## Notes

- The page uses Google Fonts for Arabic and display typography.
- The layout is designed for RTL text and includes theme/audio toggles.
